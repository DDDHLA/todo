# 待办事项应用

一个使用 Electron 开发的本地存储待办事项 macOS 应用，无需服务器，数据永久保存在本地。

## ✨ 特性

- 🎨 **现代化左右布局设计** - 左侧边栏显示统计和筛选，右侧主内容区显示待办列表
- 🌓 **明暗模式切换** - 支持亮色和暗色主题，主题设置自动保存
- 💾 **本地数据持久化存储** - 数据保存在本地，永不丢失
- ✅ **完整的待办管理** - 添加、编辑、删除、标记完成
- 🎯 **智能筛选** - 快速切换查看全部/待完成/已完成任务
- 📊 **实时统计** - 可视化展示任务统计数据
- 🚀 **无需联网** - 完全本地运行，保护隐私

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

或使用 yarn:

```bash
yarn install
```

### 开发模式运行

```bash
npm start
```

或

```bash
npm run dev
```

### 打包为 DMG 应用

```bash
npm run build:mac
```

打包完成后，DMG 文件会生成在 `dist` 目录下。

## 📁 数据存储

### 待办数据
应用数据存储在系统的用户数据目录中：
```
~/Library/Application Support/todo-app/todos.json
```

### 主题设置
主题偏好保存在浏览器的 localStorage 中，应用重启后会自动恢复

即使卸载应用后重新安装，待办数据仍然会保留。如需完全清除数据，请手动删除上述 JSON 文件。

## 🎨 功能说明

### 主题切换
点击左上角的太阳/月亮图标切换明暗模式，主题偏好会自动保存

### 添加待办事项
在右侧输入框中输入内容，按回车键或点击"+"按钮

### 完成待办事项
点击待办事项前的复选框标记完成/未完成

### 编辑待办事项
双击待办事项文本进行编辑，按回车保存，按 ESC 取消

### 删除待办事项
鼠标悬停在待办事项上，点击右侧的删除按钮

### 筛选待办事项
使用左侧边栏的筛选按钮切换视图：
- **全部任务** - 显示所有待办事项
- **待完成** - 仅显示未完成的任务
- **已完成** - 仅显示已完成的任务

### 查看统计
左侧边栏实时显示：
- 📄 全部任务数量
- ⭕ 待完成任务数量  
- ✅ 已完成任务数量

### 清除已完成
点击左侧底部"清除已完成"按钮，批量删除所有已完成的事项

## 🛠️ 技术栈

- JavaScript - 前端逻辑
- HTML/CSS - 界面设计
- Node.js - 本地文件存储
